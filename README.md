# Nifty
A discord music bot based on JDA, JDA-utilities and lavaplayer.

Nifty commands:

  Amusement:

$cat - show a random cat<br>

  Music:

$play <title|URL|subcommand> - plays the provided track<br>
$playall <URL> - plays all tracks from the playlist provided<br>
$quit - let the bot quit from playing<br>
$skip - skip to the next track in queue<br>
$nowplaying - show the track is now playing<br>
$pause - pause playing / continue playing<br>
$shuffle - shuffle the queue<br>
$repeat - set player to repeating<br>
$restart - if playing, replay current track, if not playing, replay the previous track<br>
$list - show the current queue<br>
