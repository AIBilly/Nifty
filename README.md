# Nifty
A music bot based on JDA and JDA-utilities. Will add karaoke function in future.

Nifty commands:

  Amusement:

$cat - show a random cat

  Music:

$play <title|URL|subcommand> - plays the provided track
$playall <URL> - plays all tracks from the playlist provided
$quit - let the bot quit from playing
$skip - skip to the next track in queue
$nowplaying - show the track is now playing
$pause - pause playing / continue playing
$shuffle - shuffle the queue
$repeat - set player to repeating
$restart - if playing, replay current track, if not playing, replay the previous track
$list - show the current queue
